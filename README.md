## Setup

1.  Clone the project to GOPATH

    ```sh
    git clone https://github.com/blockfint/benchmark-tm
    ```

2.  Install dependency

    ```sh
    dep ensure
    ```

3.  Run via shell script (For development)

    ```sh
    ./run.sh
    ```

3.  Test

    ```sh
    go test ./test/ -v
    ```
